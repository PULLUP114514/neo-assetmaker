# 用户手册

本手册对应当前仓库中的 `v2.1.3` 代码。版本变更和功能增量请查看 [CHANGELOG.md](CHANGELOG.md)。

## 快速开始

### 运行环境

- Windows
- Python 3.10+
- `uv`
- Rust 工具链仅在编译模拟器或完整打包时需要

### 启动应用

```bash
uv sync --no-install-project
uv run python main.py
```

首次启动会显示欢迎说明。若启用了临时项目功能，应用会在启动后直接创建一个可编辑的临时项目。

## 项目工作流

### 新建、打开与保存

- `Ctrl+N`：新建项目
- `Ctrl+O`：打开现有项目
- `Ctrl+S`：保存到当前路径
- `Ctrl+Shift+S`：另存为
- 文件菜单提供最近打开列表，最多保留 10 条记录

### 临时项目

- 启动时可自动创建临时项目，便于直接开始编辑
- 临时项目第一次保存时会走“另存为”流程
- 临时目录中的工作文件会迁移到目标目录

### 自动保存与崩溃恢复

- 自动保存服务会把备份写到项目目录下的 `.autosave/`
- 备份文件命名为 `autosave_YYYYMMDD_HHMMSS.json`
- 当前实现使用 5 分钟间隔，并最多保留 5 份备份
- 异常退出后，下次启动会检查恢复记录并弹出恢复对话框

## 素材编辑

### 基础模式与高级模式

- 基础模式适合快速配置循环素材
- 高级模式提供完整配置面板，包括基本信息、视频配置、过渡效果和叠加 UI

### 支持的素材类型

- 视频：`mp4`、`avi`、`mov`、`mkv`、`webm`、`flv`
- 图片：`png`、`jpg`、`jpeg`、`gif`、`bmp`、`webp`

### 导入与预览

- 可将视频或图片直接拖入预览区域
- 可将 `.json` 配置文件拖入右侧 JSON 预览面板进行导入
- JSON 面板是只读预览，不支持直接在面板内编辑

### 预览区快捷键

- `Space`：播放 / 暂停视频
- `←`：上一帧
- `→`：下一帧
- `W` `A` `S` `D`：微调裁剪框位置，每次移动 10 像素

这些按键需要预览区获得焦点后才会生效。

### 预览、模拟器与导出的一致性

- 入场视频和循环视频各自维护独立的时间轴 in/out、裁剪框和旋转角度
- “模拟预览”会读取当前预览页已经调整好的裁切、旋转和时间轴范围，而不是只读取原始文件
- 导出时会沿用与预览相同的入场/循环 trim 范围，入场视频不再默认按整段导出
- 循环素材如果使用图片模式，模拟器会先按当前预览状态生成临时预览视频，再进入 Rust 模拟器

### 核心配置项

- 基本信息：UUID、名称、描述、屏幕规格、图标
- 循环素材：支持视频模式和图片模式
- 入场素材：可选启用入场视频
- 过渡效果：`none`、`fade`、`move`、`swipe`
- 叠加 UI：
  - `none`
  - `arknights`
  - `image`

`arknights` 叠加模式支持干员名称、代号、条码、职业图标、颜色，以及左上角和右上栏自定义文字。

### 截帧

- 可从视频中截取当前帧并保存为图标
- 截帧适合快速生成项目图标或叠加素材

## 校验、导出与一键上传

### 校验配置

- `Ctrl+T`：手动校验配置
- 右侧 JSON 预览面板会同步显示校验状态、错误数和警告数

### 导出素材

- `Ctrl+E`：导出当前素材
- 导出前会先校验配置
- 校验失败时无法完成导出
- 入场视频和循环视频都会应用各自当前的时间轴 in/out 范围

### 一键上传

- 配置面板提供“一键上传”按钮
- 该流程会先执行导出，再通过 EPass RNDIS HTTP API 上传到设备
- 可在设置中控制上传完成后是否自动重启远端 `DrmApp`

## 远程管理

远程管理页用于连接设备、同步素材和处理远端文件。

### 使用前准备

使用前请连接通行证设备，并确认 Windows 中出现
`EPass RNDIS Remote NDIS Compatible Device` 网卡。软件会固定访问
`http://192.168.137.2/`，素材目录由设备端 HTTP API 管理为 `/assets`。

设置页提供“检测设备”按钮，可检查 RNDIS 网卡、到 `192.168.137.2` 的路由以及
`/api/v1/health` 接口状态。“上传完成后自动重启 DrmApp”仅在设备端支持
`/api/v1/drm/restart` 时生效。

### 页面功能

- 连接 / 断开设备
- 刷新远程素材列表
- 上传本地素材
- 重启远端 `DrmApp`
- 查看实时画面（需要设备端提供 HTTP MJPEG 接口）
- 查看操作日志

### 远程素材列表

连接成功后，中间列表会显示远程素材详情。每个条目提供：

- 下载
- 删除
- 编辑

“编辑”入口用于把远程素材拉回本地继续处理。

## 设置页

设置页当前分为以下几组：

- 应用设置：自动检查更新、检查频率
- 界面设置：主题、主题色、背景图、缩放、语言
- 个性化设置：临时项目、欢迎对话框、状态栏、自动保存
- 视频与导出：硬件加速预览
- 网络设置：GitHub 加速、代理
- 自动上传：EPass RNDIS 连接检测和远程上传配置
- 关于：快捷键帮助、检查更新、版本信息

## 快捷键总表

- `Ctrl+N`：新建项目
- `Ctrl+O`：打开项目
- `Ctrl+S`：保存项目
- `Ctrl+Shift+S`：另存为

## 故障排查

### 应用无法启动

- 先确认依赖已安装完成
- 确认 `uv run python main.py` 能在当前环境执行
- 若是打包版本，检查应用目录下的日志和崩溃文件

### 视频无法预览或导出

- 检查输入文件格式是否受支持
- 确认视频文件未损坏
- 若使用打包版本，检查 FFmpeg 相关依赖是否随安装包完整分发

### RNDIS 远程上传失败

- 确认设备已通过 USB 连接，并出现 EPass RNDIS 网卡
- 确认浏览器可访问 `http://192.168.137.2/api/v1/health`
- 确认 health 响应包含 `status`、`device`、`protocol_version`、`device_id` 和 `capabilities`
- 确认设备端 HTTP API 支持素材上传接口
- 如启用了自动重启，确认设备端支持 `/api/v1/drm/restart`

### 查看日志

- 应用会生成 `app_YYYYMMDD.log`
- 日志目录优先使用应用目录下的 `logs/`
- 若应用目录不可写，会回退到其他可写目录
